# Amazon Clone (HTML & CSS)

This is a simple Amazon homepage clone built using HTML and CSS.
It is a beginner frontend project created to practice layout design and styling.

 Features

* Navigation bar similar to Amazon
* Product sections and images
* Clean layout using CSS
* Basic UI design

 Tech Stack

* HTML
* CSS



Learnings 

* Structuring web pages using HTML
* Styling using CSS
* Using Flexbox/Grid for layout
* Organizing project files properly



 Future Improvements

 Make the website responsive
 Add JavaScript for interactivity
 Improve UI design and animations
